# Smart Hostel Management System

Full-stack MERN application for logging student complaints and tracking mess performance.

## Structure
- **/backend**: Node.js & Express API connected to MongoDB.
- **/frontend**: React SPA built with Vite and animated via Framer Motion.

## Quick Start

### 1. Start Backend
```bash
cd backend
npm install
node index.js
```

### 2. Start Frontend
```bash
cd frontend
npm install
npm run dev
```
